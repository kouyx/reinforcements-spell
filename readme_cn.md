# 火线增援魔法

## 描述
**火线增援** 是一个 3 级火系魔法，用于替代 **伪装 (Disguise)** 魔法。

该魔法允许英雄在冒险地图上直接从城镇驻军中派遣部队到自己的军队。

## 魔法机制

### 火系魔法不同等级效果
- **无火系魔法**：可以从**最近**城镇的驻军中派遣部队，每天可使用 1 次。
- **初级火系魔法**：可以从**最近**城镇的驻军中派遣部队，每天可使用 2 次。
- **中级火系魔法**：可以从**任意**城镇的驻军中派遣部队，每天可使用 3 次。
- **高级火系魔法**：可以从**任意**城镇的驻军中派遣部队，每天可使用 4 次。

### 使用规则
- 英雄只能召唤属于同一玩家的城镇中的生物
- 每日使用上限以英雄为单位独立计算（每天重置）

## 安装方法
1. 通过 MOD 浏览器下载，或直接将模组文件解压至《英雄无敌 3》的 ERA/Mods/ 安装目录下
2. 在模组管理器中启用该模组
3. 开始新游戏即可使用该魔法（译者注：还需要确保 wog 选项/其他 mod 等没有禁用魔法伪装大法）

## 兼容性
- 需要 Heroes III ERA 与 ERA ERM Framework
- 与大多数 ERA 模组兼容，包含三阶升级-重生 & 进阶职业。
